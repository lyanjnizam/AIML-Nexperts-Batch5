# AIML-Nexperts-Batch5
Source of information. Come here to revise the classes. And access class learning materials. 




1. Fork the main repository
    - add your name to the fork name as well
2. Go to your repositories
3. Open your forked aimlnexpertsbatch5 folder or repo
4. Copy the >  Local > https link
5. Download and install Git + VS Code
6. Open Command Prompt
    - you will see:
        - > C://Users/username> 
    - open a folder you wish to work in using
        - cd <Folder Name> e.g. cd Desktop
        - > C://Users/username/Desktop>
7. Clone the git repository
    - > C://Users/username/Desktop> git clone <https link of forked repo>
    - you have created a cloned copy of your forked repo on your own PC
8. Now you go to the folder, right click and open with VS Code
    - if that does not work
    - Open VS code directly
        - Go to explorer 
        - Open folder
        - Find the repo folder that you downloaded
        - select it
9. Go to source control
    - Find the Graph part
    - Click on Fetch from all remotes
    - this will sync your repo with the main repo

10. Create a folder of your own name in `Practice` folder
